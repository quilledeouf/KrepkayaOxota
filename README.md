# 🦌 Крепкая Охота

> Сайт для охотников и рыбаков **Ростовской области**: интерактивная карта по
> регионам/угодьям с доступной к охоте и вылову дичью/рыбой и переключением
> сезонов.

**Ценность одним предложением:** «Крепкая Охота» позволяет охотнику и рыбаку
Ростовской области за 30 секунд узнать, *что, где и когда* сейчас можно легально
добыть — в одном месте, вместо поиска по разным картам, приказам и форумам.

## ✨ Возможности (страницы)

| Страница | Что внутри |
|----------|-----------|
| **Главная** | Hero, «Место дня», карточки проверенных мест рядом |
| **Карта** | Спутниковая карта (Esri), фильтры по сезону/типу/району, маркеры мест с попапами |
| **Справочник** | Вкладки Рыбы / Животные / Птицы, поиск, сроки по сезонам, таксы |
| **Законы** | Нерестовые запреты, нормы вылова, таблица такс ущерба, ответственность |
| Гиды, Профиль | Заглушки — следующий заход |

> **Scope (решение команды):** прорабатываем **Ростов-на-Дону** — реальные места
> рыбалки/охоты, сгруппированные по районам города. Данные актуализированы по
> открытым источникам (нерест 01.04–31.05; таксы — Пост. № 1321; КоАП ст. 8.37,
> УК ст. 256) и носят ознакомительный характер.

**Дизайн** выполнен по гайдлайнам **UI/UX Pro Max** (шрифт Inter, SVG-иконки,
hover/focus-состояния, адаптив, `prefers-reduced-motion`) с сохранением стека
HTML/CSS/JS. Анимации — нативные CSS (без Framer Motion/React).

## 🛠 Технологический стек

| Слой | Технология | Почему |
|------|-----------|--------|
| **Frontend** | HTML5 / CSS3 / Vanilla JS (ES6) | Решение команды; без сборки — легко запускать и проверять |
| **Карта** | [Leaflet.js](https://leafletjs.com/) + Esri World Imagery | Бесплатно, спутниковая подложка, маркеры и попапы |
| **Данные (MVP)** | `js/data.js` (встроенные) | На прототипе бэкенда нет — данные на фронте, доступ через `api.js` |
| **Backend (этап 2)** | Node.js + Express | Один язык с фронтом, низкий порог входа |
| **БД (этап 2)** | PostgreSQL + PostGIS | Реляционные данные + геозапросы по угодьям |

> На MVP сайт **полностью статический** и работает без сервера приложений.
> Реальный REST API спроектирован «на будущее» — см.
> [docs/API.md](docs/API.md) и [docs/ARCHITECTURE.md](docs/ARCHITECTURE.md).

## 🚀 Как запустить

> Нужен только браузер. Интернет нужен для подложки карты (спутник Esri), шрифта
> Inter и фото мест; без сети интерфейс и данные продолжают работать.

### Вариант 1 — локальный сервер (рекомендуется)

Из корня репозитория:

```bash
# Python 3 (есть почти везде)
cd frontend
python -m http.server 5500
```

Открыть в браузере: **http://localhost:5500**

Альтернатива на Node.js:

```bash
npx serve frontend     # или: npx http-server frontend -p 5500
```

В VS Code удобно расширение **Live Server** → правый клик по
`frontend/index.html` → *Open with Live Server*.

### Вариант 2 — просто открыть файл

Прототип сделан так, что данные встроены в `js/data.js`, поэтому
`frontend/index.html` можно **открыть двойным кликом** (file://) — карта и
заглушки заработают без сервера.
